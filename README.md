# dotfiles

Personal development environment.

## Contents

- .zshrc
- .gitconfig
- .gitignore_global
- .tool-versions
- Brewfile

## Setup

```bash
brew bundle
cp .zshrc ~/.zshrc
cp .gitconfig ~/.gitconfig
cp .gitignore_global ~/.gitignore_global
cp .tool-versions ~/.tool-versions
source ~/.zshrc
```
