# GitHub CLI Commands

## Repository

```bash
gh repo clone Osk0830/debug-handbook
gh repo view --web
gh repo list Osk0830
```

## Pull Request

```bash
gh pr status
gh pr create
gh pr view --web
gh pr checkout <number>
gh pr merge
```

## Issue

```bash
gh issue list
gh issue create
gh issue view <number> --web
```

## Workflow

```bash
gh run list
gh run view
gh run watch
```
