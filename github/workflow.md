# GitHub CLI Workflow

## よくある作業フロー

### リポジトリをclone

```bash
gh repo clone Osk0830/debug-handbook
```

### PRを確認

```bash
gh pr status
```

### PRを作成

```bash
gh pr create --web
```

### GitHubをブラウザで開く

```bash
gh repo view --web
```

## dotfilesでの使い方

GitHub CLIの認証情報は管理しません。

管理するのは以下だけです。

- 認証手順
- よく使うコマンド
- aliasの候補
